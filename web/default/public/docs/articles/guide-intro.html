goodbyeri.cc 使用文档

goodbyeri.cc 是新一代 AI 模型网关,用一个统一的 API 入口接入 Claude、OpenAI、Gemini 等多家大模型,内置限流、令牌管理、计费与多语言支持。你可以把它无缝接入 Claude Code、Codex CLI、Gemini CLI 以及 VSCode / Cursor 等工具。

三步接入

1. 在 goodbyeri.cc 注册并创建 API 密钥;
2. 将工具的 BASE_URL 指向 goodbyeri.cc(Claude 用 https://goodbyeri.cc,Codex 用 https://goodbyeri.cc/v1);
3. 填入你的 API 密钥,开始使用。
